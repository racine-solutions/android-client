# RacinePay Quick Guide: Login + Tenant Setup

Use these 3 steps to configure tenant and sign in.

## Step 1: Configure Tenant

1. Open the app.
2. Tap **Update Server Configuration** on the login screen.
3. In **Server Config**, update only **Tenant** (other fields are locked).
4. Tap **UPDATE CONFIG** and let the app restart.

## Step 2: Login

1. Re-open the app.
2. Enter `Username` and `Password`.
3. Tap **Login**.
4. Complete passcode flow (setup or enter existing passcode).

## Step 3: Change Tenant Later (Optional)

1. Open side menu -> **Settings** -> **Server Config**.
2. Update **Tenant**.
3. Tap **UPDATE CONFIG** and restart when prompted.

## Notes

- Config save logs out current session and restarts the app.
- Tenant change applies after restart.
- If login fails, first verify tenant + credentials with your admin team.
